## 기본 동작

### 색인

#### _id 지정 색인

```yaml
PUT [인덱스 이름]/_doc/[_id값]

{
[문서 내용]
}
```

#### _id를 지정하지 않고 색인

```yaml
PUT [인덱스 이름]/_doc

{
[문서 내용]
}
```

### 조회 / 검색

#### 문서 조회

```yaml
GET [인덱스 이름]/_doc/[_id값]
```

#### 문서 검색

```yaml
GET [인덱스 이름]/_search # REST API 관점에서 GET에 요청 본문 작성에 대한 이견이 많았음
POST [인덱스 이름]/_search # 각자의 상황과 철학에 맞는 메서드를 사용하면 됨

{
[쿼리]
}
```

### 업데이트

#### 문서 업데이트

```yaml
POST [인덱스 이름]/_update/[_id값]

{
"doc": {
[문서 내용]
}
}
```

### 삭제

#### 문서 삭제

```yaml
DELETE [인덱스 이름]/_doc/[id값]
```
